</p>

## Development setup

### Installation

- Install [nvm](https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating)

- Install node

```sh
nvm install
```

- Install packages

```sh
npm install
```

- Create a `.env` file and copy the content of `.env.example` to it.

```sh
cp .env.example .env
```

### Usage

- Start development server

```sh
npm run dev
```

- Open http://localhost:5173
